# StudyGenie 📚🤖

StudyGenie is an AI-powered mobile study assistant that helps students manage their academic life efficiently.

## Features
- 📅 Smart class routine notifications
- 📖 Notes upload for remote study
- 🧠 AI doubt solving
- ⏱ Focus study sessions
- 📊 Study progress tracking

## Tech Stack
- React Native
- Expo
- TypeScript
- AI APIs

## Installation

```bash
git clone https://github.com/sankrittx/studygenie.git
cd studygenie
npm install
npx expo start
